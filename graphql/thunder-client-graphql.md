# Thunder Client GraphQL API

Thunder Client is the flagship VS Code REST API client extension offering a lightweight GUI for sending HTTP requests, managing collections with environment variables, and running scriptless tests. Features include Git Sync for team collaboration, GraphQL support, pre/post request scripting, collection import from Postman/Insomnia/Hoppscotch/OpenAPI, and an advanced CLI supporting CI/CD pipelines.
